# 아이뜨락 어린이집 — Premium Renovation v2.0

> 북유럽 감성의 프리미엄 보육 공간

## 프로젝트 정보

- **타입**: Progressive Web App (PWA)
- **목적**: 어린이집 소개 및 상담 신청
- **디자인**: Scandinavian Warm Premium
- **기술**: HTML5 + Tailwind CSS + Vanilla JS + AOS + Swiper.js
- **배포**: Vercel

## 주요 기능

✅ 프리미엄 로딩 스크린  
✅ 북유럽 감성 디자인 시스템 (Sage/Terracotta/Ocean)  
✅ AOS 스크롤 애니메이션  
✅ 커스텀 커서 (데스크톱)  
✅ 다크 모드 (토글 + 시스템 감지 + localStorage)  
✅ 어린이집 소개 (카운터 애니메이션)  
✅ 교육 철학 섹션  
✅ 6개 교육 프로그램 카드  
✅ 4명 교사진 프로필  
✅ 6개 시설 호버 리빌 카드  
✅ 하루일과 타임라인  
✅ 학부모 후기 Swiper 캐러셀  
✅ FAQ 아코디언 (6문항)  
✅ 상담 신청 폼 + 성공 모달  
✅ 풀스크린 모바일 메뉴  
✅ 트러스트 마키 밴드  
✅ 스크롤 프로그레스 바  
✅ 플로팅 액션 버튼 (전화/카카오톡/맨위로)  
✅ 뉴스레터 구독 폼  
✅ PWA 지원 (앱 설치 가능)  
✅ 반응형 디자인 (모바일 퍼스트)  
✅ 접근성 (키보드, 모션 감소, 포커스)  
✅ 프린트 스타일

## 빠른 시작

```bash
npx serve .          # 로컬 개발
vercel --prod        # 프로덕션 배포
```
