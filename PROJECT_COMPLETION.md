## 🛠️ 기술 스택
### Frontend
- **HTML5**: 시맨틱 구조, SEO 최적화
- **Tailwind CSS**: 유틸리티 우선 CSS 프레임워크 (CDN)
- **Vanilla JavaScript**: 고성능 인터랙티브 기능
- **Font Awesome**: 아이콘 시스템
- **Google Fonts**: Noto Sans KR, Fredoka 폰트

### 특수 기능
- **Glass Morphism**: 유리 효과 디자인
- **3D Transforms**: perspective 기반 카드 애니메이션
- **Particle System**: 커스텀 파티클 애니메이션
- **Intersection Observer**: 스크롤 기반 애니메이션
- **Local Storage**: 다크모드 상태 저장

## 🚀 주요 기능들
### 1. 로딩 스크린
- 브랜드 아이덴티티 강화
- 부드러운 페이드 아웃 효과
- 1.5초 최적화된 로딩 시간

### 2. 파티클 시스템
- 50개 파티클 동시 애니메이션
- 6초 사이클, 랜덤 지연시간
- 성능 최적화된 CSS 애니메이션

### 3. 3D 카드 효과
- perspective: 1000px 3D 공간
- 호버 시 rotateY(5deg) rotateX(5deg)
- 부드러운 0.6초 transition

### 4. 다크모드 시스템
- localStorage 상태 저장
- 완벽한 색상 대비 (WCAG 준수)
- text-shadow로 가독성 극대화

### 5. 스크롤 진행 바
- 실시간 스크롤 진행도 표시
- 무지개 그라데이션 색상
- 부드러운 0.3초 transition

### 6. 카운터 애니메이션
- Intersection Observer 기반
- 200단계 부드러운 카운팅
- 섹션 진입 시 자동 실행

## 📱 반응형 디자인
### 브레이크포인트
- **Mobile**: < 768px (기본)
- **Tablet**: md: 768px+
- **Desktop**: lg: 1024px+
- **Large**: xl: 1280px+

### 반응형 특징
- Grid 시스템: 2→3→4 컬럼 자동 조정
- 텍스트 크기: text-5xl → md:text-7xl
- 간격 조정: gap-4 → gap-8
- 메뉴: 모바일 햄버거 → 데스크톱 수평
